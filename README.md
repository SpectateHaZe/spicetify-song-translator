# Song Title Romanizer & Translator

A Spicetify extension that translates foreign song titles (Japanese, Chinese, Korean, Cyrillic, etc.) and provides Romanization (Romaji/Pinyin/etc.) in the now-playing bar and lists globally.

## Features
- **Player Bar:** Displays the Romanized title and Translation stacked neatly below the song title.
- **Zero API lag:** Uses a local persistent cache to prevent hitting API rate limits.
